
# CaseOpening

New casino game for "Gamble With Your Friends"

Open as many weapon cases as you can, go for gold, but keep your eyes peeled for a blue gem.

Installation:
Note: Make sure every playing in the lobby has this mod downloaded.
1. Install BepInEx 5.x into your Gamble With Your Friends directory.
2. Copy the "Games" folder into: BepInEx/plugins/CaseOpening/
3. Ensure the "Games" folder has a "CaseOpening" folder with an asset bundle, dll and gamemanifest.json
